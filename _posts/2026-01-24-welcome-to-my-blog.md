## About This Blog

This blog is built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/). It features a clean, minimalist design that puts the focus on content.

### Key Features

Here are some of the features I've implemented:

- **Minimalist Design**: Clean typography and generous whitespace for comfortable reading
- **Tags & Categories**: Easy content organization and discovery
- **Comments**: Powered by [Giscus](https://giscus.app/), using GitHub Discussions
- **Syntax Highlighting**: Beautiful code blocks for technical posts
- **Responsive**: Works great on all devices

## How It Works

Adding a new blog post is simple. I just create a new Markdown file in the `_posts` directory with the naming convention `YYYY-MM-DD-title.md`, add some frontmatter, and commit it to GitHub. Here's an example:

```markdown
---
layout: post
title: "My New Post"
date: 2026-01-24
categories: [tutorial]
tags: [web-development, jekyll]
---

Your content goes here...
```

Within a few minutes, GitHub Pages automatically builds and deploys the updated site. No manual build process, no deployment scriptsâ€”just push and publish.
